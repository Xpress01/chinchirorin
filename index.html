Rules of game

  • Player and pc both roll 3 dice
  • If player/pc roll doubles, that is considered their score
  • The player/pc with the highest score wins
  • If no one rolls a double, it's a draw
  • If player rolls a 4,5,6 or triple numbers except for 1,1,1 it is an automatic win
  • If player rolls a 1,2,3 or 1,1,1 it is an automatic loss
